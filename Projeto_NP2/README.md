## Uso de IA

> **Modelo:** ChatGPT baseado no GPT-5.3
>
> **Fornecedor:** OpenAI

Este projeto contou com o auxílio de inteligência artificial como apoio didático para esclarecimento de dúvidas conceituais, discussão de decisões de implementação e correção de bugs durante o desenvolvimento.

Nenhum código foi copiado diretamente, todo código do projeto foi escrito manualmente do início ao fim de seu processo de criação.

### Prompts Utilizados (exemplos)

```
"Chat, por que o printf está imprimindo muitos zeros após a vírgula?"
```

**Resultado:** Satisfatório. Permitiu ajustar a formatação com %.2f ou %.0f.

```
"Chat me passa uma lista de principais formatações do printf?"
```

**Resultado:** Satisfatório. Permitiu o uso de formatações para impressão de diferentes tipos de variáveis.

```
"Chat, o que esse erro significa? ..."
```

**Resultado:** Satisfatório. Permitiu a correção de diferentes erros de sintaxe presentes na fase final do projeto.

```
"Chat durante toda essa conversa evite me dar exemplos diretos de códigos escritos em java."
```

**Resultado:** Satisfatório. Além de impulsionar o meu aprendizado, me ajudou a tratar conceitos e a otimizar o projeto sem me oferecer códigos para serem copiados diretamente.

```
"Como posso implementar um sistema de cadastro utilizando arquivos?"
```

**Resultado:** Satisfatório. Auxiliou na definição da estratégia de armazenamento de usuários e conteúdos utilizando arquivos CSV.

```
"Como aplicar herança, classes abstratas e interfaces neste projeto?"
```

**Resultado:** Satisfatório. Auxiliou na compreensão e aplicação dos conceitos de orientação a objetos utilizados na modelagem do sistema.

```
"O que são threads e como poderiam ser utilizadas no projeto?"
```

**Resultado:** Satisfatório. Contribuiu para o entendimento do conceito de execução concorrente e suas possíveis aplicações no sistema.

## Conclusão

A inteligência artificial foi utilizada apenas como ferramenta de estudo e apoio ao desenvolvimento. Também foram utilizados como referência os materiais disponibilizados pelo professor Christopher Lima e resoluções de exercícios realizados anteriormente durante o semestre.

A ferramenta contribuiu principalmente para o esclarecimento de conceitos, correção de erros, compreensão de funcionalidades da linguagem Java e discussão de alternativas de implementação, sem substituir o processo de desenvolvimento realizado pelos autores.
